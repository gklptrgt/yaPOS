# Features
- Customise menu items with custom colors.
- Table system
- Express Sale (for non-table sales fast)
- Printer and Scanner
- Generate Reports
- Add items, invoices, companies
- Stock System
- Notification system for; low stock, high sales
- Barcode design and printing
- ML for finding relations between your products, create sales or change location.
- ML for predicting best prices, profits, expenses.
